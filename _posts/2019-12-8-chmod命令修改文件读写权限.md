## chmod命令使用


## 使用规则

```java
 chmod [ u / g / o / a ] [ + / - / = ] [ r / w / x ] file
```

其中的参数意义：

* `u `代表User，是文件所有者

* `g`代表和User同Group的用户

* `o`代表Other，其他用户

* `a`代表ALL，所有用户

* `+`代表增加权限

* ` -`代表取消权限

* `=`代表取消之前的权限，并给唯一权限

* `r`代表Read，只读文件

* `w`代表Write，写文件

* `x代表运行文件

* `file` 代表文件路径

举例

```java
chmod  a+r  hello.txt
```
